Animator

public final class Animator<T> where T: Animation

Runs an animation until it is either finished or cancelled. Animators cannot be reused:

  • They begin in a ‘running’ state
  • If the animation finishes, the animator enters the completed (finished) state.
  • If cancel() is called on the animator while in a running state, the animator enters the completed (cancelled) state.
