  • close(_:)

    Closes the event.

    If the event has already been closed, this has no effect.

    Calls all observers with the given payload. Once an event is closed, calling fire(payload:) or close(payload:) will have no effect. Adding an observer after an event is closed will simply call the observer synchronously with the payload that the event was closed with.
